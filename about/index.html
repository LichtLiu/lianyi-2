1 Introduction — what are Git and GitHub?

Git

  • Version control system (track changes in folder/files)
  • Snapshots stored in an “invisible” .git folder
  • a history of snapshots of a file system

Install GIT

  • on macOS: open a terminal and type git --version . This will prompt an installer if you don’t have it already
  • on Windows: download the installer from https://git-scm.com/download/win
  • *(see https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
